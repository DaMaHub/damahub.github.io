Collective Intelligence - Network Library

A 21st century society requires new ways of harnessing and organising collective intelligence. DaMaHub is building tools to allow peers and librarians to create and manged knowledge bases that evolve over time through networked libraries.

Data

Data structure, types, stores and infrastructure is the base layer to any evidence base. DaMaHub builds on existing and the latest peer to peer crypto protocols.

LibraryToolkit

The DaMaHub toolkit provides tools to build and evolve evidence bases, including: indexing, data packaging and rules on what to keep and preserve.

Governance

Use decentralised autonomous organisation (DAO) to apply rules over voting and standard settings.
